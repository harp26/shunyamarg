How to import the ShunyaMarg CSV files

1. Download the CSV file from Claude (e.g. tattvabodh_section6.csv)
2. Click Choose File above and select the downloaded CSV
3. Click Import →
4. Cards appear in the Cards panel — check status and sort order

Note: File upload handles Sanskrit, Devanagari and special characters correctly. Paste method may corrupt these characters — use file upload for ShunyaMarg content.
